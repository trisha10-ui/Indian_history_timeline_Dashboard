Project References and Data Source

This page details the tools and data sources used to construct this static dashboard.

Technology Used:

  • **MS-Excel:** Used for primary data storage and structure (exported as `timeline_data.csv`).
  • **HTML5:** Core structure of all web pages.
  • **CSS3:** Custom styling for the timeline structure and era-specific colors.
  • **Bootstrap 5.3:** Used for responsive layout, navigation, cards, and tabs.
  • **Vanilla JavaScript (Client-Side):** Used to read the local CSV file and dynamically generate the timeline on `index.html`.

Data Source Note:

The data points used in the timeline were compiled from general knowledge sources on Indian History. Dates (especially those prefixed with 'c.') are approximate and representative of the beginning of the respective period or event.
